# folder 세팅
1. 폴더 이름은 반드시 영문으로 만든다. (한글이나 기타 다른 언어로 작성할 경우 문제가 생긴다. -웹에서 작동이 안 될 수 있다.)
2. 첫글자를 숫자로 사용하지 않는다. 무조건 영문으로.
3. 폴더 및 파일은 띄어쓰기를 사용하지 않는다. (_ 혹은 - 으로 )
4. 특수문자 역시 사용하지 않는다.

---

## 쿠팡기준 : 폴더, 파일, 문서내 이름 : 단어/형용사
